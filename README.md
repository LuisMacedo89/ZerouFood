# ZerouFood — Etapa 5: Publicação real

Esta etapa prepara o projeto para sair do protótipo e virar um MVP publicável.

## Arquivos incluídos

- `index.html` — marketplace/página inicial
- `produto.html` — página de produto
- `painel-vendedor.html` — painel do vendedor
- `admin.html` — painel administrativo inicial
- `supabase-schema.sql` — estrutura inicial do banco de dados
- `checklist-publicacao.md` — passo a passo para colocar no ar

## Caminho recomendado

1. Criar conta no Supabase.
2. Criar projeto `zeroufood`.
3. Rodar o arquivo `supabase-schema.sql`.
4. Criar bucket de imagens chamado `product-images`.
5. Criar conta na Vercel.
6. Subir esses arquivos para um repositório GitHub.
7. Importar o repositório na Vercel.
8. Conectar domínio próprio.
9. Depois substituir o `localStorage` por Supabase.

## Observação importante

Este pacote ainda é um MVP estático/preparatório.  
A publicação real com login, banco e upload definitivo exige conectar o código ao Supabase.
